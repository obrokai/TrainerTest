# 遠距互動健身教練系統構想

本專案旨在規劃一套透過鏡頭互動的健身教練與學生系統，讓教練能於視訊畫面中在空中繪製方向線與提示，學生可即時接收動作指導與角度建議。

## 系統目標
- 讓教練能在視訊畫面上即時比劃方向、角度與修正建議。
- 協助學生藉由教練的提示與 AI 動作分析得到即時的動作回饋。
- 支援遠距一對一與小班制課程。

## 核心功能
1. **即時雙向視訊串流**：採用 WebRTC 或類似技術，確保低延遲、高畫質。
2. **空中筆劃與標註**：教練可透過手勢或簡易控制器在畫面上畫線、標記角度，並同步顯示給學生。
3. **姿勢偵測與動作評估**：結合電腦視覺模型（如 MediaPipe Pose、OpenPose）進行關節定位與角度計算，提供即時回饋。
4. **語音與訊息輔助**：除視覺標註外，支援語音提示與文字訊息。
5. **課程錄製與回放**：可記錄課程影片與標註，以供課後複習。

## 架構概述
- **前端（教練端／學生端）**：Web 或行動應用程式，負責視訊串流顯示、互動畫面與手勢輸入。
- **即時通訊服務**：透過訊號伺服器協調 WebRTC 連線，並提供筆劃資料的同步機制（如 WebSocket）。
- **AI 分析服務**：負責姿勢偵測、角度計算與提示產生，可部署在雲端或邊緣裝置。
- **資料儲存**：保存課程錄影、標註紀錄、學員進度與課程內容。
