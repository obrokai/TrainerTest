# Snatch — 健身房智慧管理系統

臉部辨識 × Adaptive UI × LINE LIFF。讓進場、營運與經營，一次到位。

一個設計導向的單頁式產品官網，介紹 Snatch 健身房智慧管理系統的三大體驗：

- **會員的一天** — 臉部辨識進場、LINE 原生會員服務、兩步約課
- **櫃檯的一天** — Adaptive UI 對話式後台、主動風險提醒
- **老闆與教練的視角** — 即時 KPI、流失漏斗、月結薪資報表

## 技術

純靜態 `index.html`，無建置流程、無框架依賴。所有裝置 mockup 皆以 SVG / CSS 手刻，僅外部載入 Google Fonts。

## 部署

部署於 Vercel，根目錄即為 `index.html`，無需額外設定。
